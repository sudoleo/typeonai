ChatGPT:

Machbar nur mit einem kleinen Flex-Trick ✨

Im Moment hängen Titel + Buttons “nebeneinander”, aber nicht in einem echten Layout, das bis ganz rechts reicht. Lösung: Das h2 wird zum Flex-Container, und der Button-Block bekommt margin-left: auto;. Dann schiebt er sich automatisch an den rechten Rand der Consensus-Box.

Ändere dein CSS so:

css
/* Überschrift + Inline-Buttons */
.consensus-main h2 {
  display: flex;              /* statt inline-flex */
  align-items: center;
  gap: 6px;
}

/* Container, der direkt im H2 hängt */
.consensus-copy-inline {
  display: inline-flex;
  align-items: center;
  gap: 8px;
  margin-left: auto;          /* schiebt die Buttons nach rechts */
}

/* Buttons: transparent, lean */
.consensus-copy-icon-btn {
  border: none;
  background: transparent;
  padding: 0 4px;
  margin: 0;
  cursor: pointer;
  display: inline-flex;
  align-items: center;
  gap: 4px;
  font-size: 0.8rem;
  color: inherit;
}

.consensus-copy-icon-btn .copy-label {
  font-size: 0.8rem;
}

/* Hover-Effekt nur über Opacity/Underline, kein Background */
.consensus-copy-icon-btn:hover {
  opacity: 0.9;
  text-decoration: underline;
}

.consensus-copy-icon-btn:focus-visible {
  outline: 1px solid currentColor;
  outline-offset: 2px;
}

/* Copy-Icon: zwei überlappende Kästchen wie im Screenshot */
.copy-icon {
  position: relative;
  width: 14px;
  height: 14px;
  display: inline-block;
  border-radius: 3px;
  border: 1px solid currentColor;
  box-sizing: border-box;
}

.copy-icon::before {
  content: "";
  position: absolute;
  top: -3px;
  left: 3px;
  width: 14px;
  height: 14px;
  border-radius: 3px;
  border: 1px solid currentColor;
  opacity: 0.7;
  box-sizing: border-box;
}

Falls du magst, kann man für kleine Screens noch flex-wrap: wrap auf .consensus-main h2 setzen, dann rutschen die Buttons bei wenig Platz sauber in die nächste Zeile, bleiben aber rechtsbündig.

ChatGPT:

Klar 😄 Du kannst in deinem bestehenden @media (max-width: 768px)-Block einfach die Flex-Ausrichtung der Button-Gruppe ändern, damit sie untereinander stehen.

Pack das hier in deine Media Query:

css
@media (max-width: 768px) {
  .consensus-main h2 {
    flex-wrap: wrap;          /* erlaubt Umbrechen */
  }

  .consensus-copy-inline {
    flex-direction: column;   /* Buttons untereinander */

    gap: 4px;                 /* etwas weniger Abstand */
    margin-left: auto;        /* bleibt an der rechten Seite */
  }
}

Damit bleiben die Buttons rechts, stehen auf Mobile aber schön untereinander statt in einer Reihe.
